# cue-sandbox

A sandbox repository for learning and experimenting with [CUE](https://cuelang.org/) - an open-source language for defining, generating, and validating data.

## What is CUE?

CUE (Configure, Unify, Execute) is a language designed for:
- Configuration management
- Data validation (JSON, YAML)
- Schema definition
- Code generation

## Getting Started

```bash
# Install CUE
brew install cue-lang/tap/cue

# Verify installation
cue version
```
